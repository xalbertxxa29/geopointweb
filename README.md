# GeoPoint Web App

Futuristic web application with Firebase integration.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Features

-   **Futuristic Design**: Glassmorphism, neon accents, and dark mode.
-   **Firebase Auth**: Secure login with email/password.
-   **Dashboard**: Real-time indicators from Firestore "tareas" collection.
-   **Responsive**: Works on desktop and mobile.

## Project Structure

-   `index.html`: Login page.
-   `menu.html`: Dashboard page.
-   `src/`: Source files (JS, CSS, Firebase config).
-   `imagenes/`: Assets (logo).
